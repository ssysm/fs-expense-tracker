# restcms
A Lightweight, Simple, RESTful API CMS system

## Setup

## Dependencies
   - MongoDB v3.4+
   - NodeJS v9.3.0+
   
### Run
   
#### Run API Server
    $ npm install
    $ npm run start
API Server Listen @ port `3000` (Port can be changed in `./bin/www`)

#### Compile API Documentation

    $cd docs
    $npm install
    $npm run build
    
Compiled API Documentation @ `./docs/index.html`
